# QIQU Project - Quantum Intelligence Quotient Unit

This repository contains the planning and technological artifacts for the QIQU (Quantum Intelligence Quotient Unit) project by Hubstry DeepTech. QIQU is a conceptual and exploratory hub dedicated to monitoring, theoretical research, and building a strategic vision in the field of quantum computing, with a focus on Artificial Intelligence, cryptography, programming languages, and data infrastructure.

---

## Repository Content

### Original Modules
*   `planejamento_expansao_qiqu.md`: Document detailing the original expansion plan.
*   `docs/`: Directory for additional documentation.
*   `gurudev-qc-sdk/`: SDK (Software Development Kit) for Gurudev-QC.
*   `quantum-circuit-visualizer/`: Quantum circuit visualizer.
*   `quantum-ai-demos/`: Demo applications for Quantum AI.
*   `post-quantum-crypto-demos/`: Demos for post-quantum cryptography.
*   `qiqu-portal/`: QIQU web portal.
*   `data/`: Quantum datasets and models.

### Hubstry Quantum Integration Module
This section describes the project's expansion to integrate quantum simulation and emulation with the **Hubstry Harmonic Protocol**.

*   `quantum-simulation/`: Quantum simulation implementations (e.g., adiabatic algorithms) in **Rust**.
*   `quantum-emulation/`: Emulators to reproduce quantum phenomena (e.g., QKD) in **Rust**.
*   `harmonic-mapping/`: **Python** algorithms for mapping harmonic series to quantum states.
*   `integration-tests/`: Integration tests between the quantum layer and the Hubstry IoT protocol.
