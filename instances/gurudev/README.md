# GuruDev® — Programação Comparada (AlterLang™)

**Instância IPII original — linguagem de programação multissemiótica**

Esta instância é a aplicação fundadora da IPII:
transpilação semântica entre linguagens de programação,
com preservação de significado ontológico.

## Repositório principal

O desenvolvimento do GuruDev® ocorre em repositório dedicado
no ecossistema Hubstry Deep Tech.

## Conceitos centrais

- **DISPATCH_ON_HERMENEUTICS**: instrução central do GuruCompiler®
- **GVM**: Gurudev Virtual Machine
- **GuruByte**: formato de bytecode semântico
- **AlterLang™**: framework de programação comparada
- **GuruMatrix 5D**: base ontológica (documentada em core/IPII-SPEC.md)

## Publicação acadêmica

Whitepaper v0.2 e MVP Python com 25 testes automatizados
verificando o DISPATCH_ON_HERMENEUTICS disponíveis no Zenodo.
