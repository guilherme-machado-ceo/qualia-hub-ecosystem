# Instâncias IPII

Este diretório contém implementações da IPII em domínios específicos.

## O que é uma instância

Uma instância IPII é uma aplicação do engine semântico a um par
de sistemas formais com vocabulários distintos. O core IPII é
agnóstico de domínio; cada instância especializa os 5 eixos
paramétricos e o banco pan-domínio para seu contexto.
